The contents of this repository are for the analyses of Dendrogyra cylindrus microbiomes.

dendrogyra_16S_filtered_by_500.R is the main analyses included in the manuscript Cauvin et al., 2026

blast_db_becker_samples.R is a  the nucleotide BLAST analysis comparing D. cylindrus-associated ASVs with previously-identified SCTLD bioindicators

shared_ASVs_w_McCauley_data.R is the comparison of ASVs from this dataset with those in the McCauley et al., 2023 dataset
